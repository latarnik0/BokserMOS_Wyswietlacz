# PROJEKT - BokserMOS_Wyswietlacz
Wyświetlacz do projektu boksera z koła naukowego MOS

# CEL
Celem jest wykonanie wyświetlacza OLED lub LCD do interfejsu z przyciskami (włączanie sprzetu, odpalenie gruchy, klawiatura do wpisywania nickname gracza do rankingu)

# WYKORZYSTANY SPRZĘT
- Raspberry Pi Pico H
- Wyświetlacz LCD (20x4)
- Potencjometr liniowy
- Przyciski
- Przełącznik On/Off
- Inne: płytka stykowa, kable, przejściówka na potencjometr
