# PROJEKT - BokserMOS_Wyswietlacz
Wyświetlacz do projektu boksera z koła naukowego MOS

# CEL
Celem jest wykonanie wyświetlacza OLED lub LCD do interfejsu z przyciskami (włączanie sprzetu, odpalenie gruchy, klawiatura do wpisywania nickname gracza do rankingu).

# DZIAŁANIE
Wpisywanie nicku za pomocą potencjometru. Użytkownik steruje potencjometrem i wartość rezystancji jest zmapowana na alfabet angielski. Mamy 4 przyciski odpowiedzialne za: zapisanie aktualnie wyświetlanego znaku, kasowanie znaku, zapisywanie gotowego nicku, reset. 

# WYKORZYSTANY SPRZĘT
- Raspberry Pi Pico H
- Wyświetlacz LCD (20x4)
- Potencjometr liniowy
- Przyciski
- Przełącznik On/Off
- Inne: płytka stykowa, kable, przejściówka na potencjometr

# DODATKOWE INFORMACJE
- **Data rozpoczęcia:** październik 2025
- **Status:** w trakcie rozwoju
- **Ostatnia aktualizacja:** 20.10.2025

# DO ZROBIENIA 
- Dodać włącznik
- Dodać przycisk kasujący znak
- Zmodyfikować tryb 1 pracy - po wcisnięciu enter czekamy chwile np 5s żeby użytkownik miał czas się odsunąć, grucha się obniża
